# Mission Control

Mission Control (⌘M) is a dashboard showing all active sessions at a glance.

![Mission Control panel](../../screenshots/mission-control.png)

## Overview

Each session appears as a card showing:
- Session name and workspace
- CLI type and model
- Current status: **running** (green), **idle** (yellow), **exited** (gray)
- Recent output snippet

## Keyboard navigation

Use arrow keys to navigate between session cards. Press **Enter** to switch to a session.

## Use cases

- **Spot idle sessions** — see which sessions are waiting for input at a glance
- **Quick switching** — jump between sessions without using the sidebar
- **Status overview** — check which sessions are actively working
