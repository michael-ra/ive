# Inbox

The Inbox (⌘I) collects sessions that have exited or are waiting for your attention.

![Inbox panel](../../screenshots/inbox.png)

## What appears in the Inbox

- Sessions that have **exited** (process finished or was stopped)
- Sessions that are **idle** and have sent an oversight nudge
- Sessions waiting for user input after completing a task

## Actions

| Action | Description |
|--------|-------------|
| **Open** | Switch to the session tab |
| **Dismiss** | Remove from inbox (session stays in sidebar) |
| **Delete** | Delete the session entirely |
| **Dismiss All** | Clear all inbox items at once |

## Archive

Dismissed items move to the **Archive** view. Toggle it with the archive button to review past sessions.

## Keyboard navigation

- **↑/↓** — navigate items
- **Enter** — open the selected session
- **Delete** — dismiss the selected item

## Oversight nudges

When a session goes idle after completing work, Commander sends an oversight nudge — a notification suggesting you review the output before the session proceeds. This helps catch mistakes in autonomous workflows.
