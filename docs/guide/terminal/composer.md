# Composer

The Composer (⌘E) is a structured multi-line input editor for crafting complex prompts.

## Features

- **Bullet points** — type `→`, `-`, `*`, or `>` to start a bullet
- **Markdown headers** — type `#`, `##`, or `###` for section headers
- **Indentation** — Tab/Shift+Tab to indent/outdent bullets
- **Auto-formatting** — checked items become **bold** on submission
- **Height expansion** — the editor grows as you type

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| ⌘E | Open / close Composer |
| ⌘↵ | Submit the composed message |
| Esc | Close without submitting |
| Tab | Indent current bullet |
| Shift+Tab | Outdent current bullet |

## Use cases

- Writing multi-step instructions with numbered lists
- Composing structured prompts with sections
- Creating detailed task descriptions before sending
