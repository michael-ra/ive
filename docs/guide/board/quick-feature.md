# Quick Feature

Quick Feature (⌘⇧N) is a minimal modal for capturing task ideas instantly without switching context.

## How it works

1. Press **⌘⇧N** from anywhere in the app
2. Type a title (required)
3. Optionally add a short description
4. Press **Enter** — the task is created in the **Backlog**

## Voice input

The Quick Feature modal supports voice input via the browser's Web Speech API. Click the microphone icon and speak your task description.

## Use cases

- Capturing ideas while you're in the middle of something else
- Creating tasks from Claude's output (copy a suggestion, open Quick Feature, paste)
- Building a backlog quickly during a planning session
